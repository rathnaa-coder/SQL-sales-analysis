# SQL Sales Analysis

A beginner SQL project that analyzes sales transaction data to answer common business questions using SQL. This project demonstrates core SQL concepts such as filtering, sorting, aggregation, grouping, and basic business reporting.

---

## 📌 Project Objective

The objective of this project is to practice SQL by analyzing sales data and extracting meaningful business insights through structured queries.

---

## 🛠️ Technologies Used

- SQL (SQLite)
- DB Fiddle
- Visual Studio Code
- Microsoft Excel

---

## 📂 Dataset

The dataset contains **50 sample sales transactions** with the following fields:

- Order ID
- Order Date
- Customer Name
- Product Category
- Amount
- Payment Method

---

## 📊 SQL Concepts Demonstrated

- SELECT
- WHERE
- ORDER BY
- LIMIT
- GROUP BY
- HAVING
- SUM()
- AVG()
- COUNT()

---

## 📈 Business Questions Answered

- Display all sales records.
- Filter transactions by product category.
- Find transactions above a specified amount.
- Identify the highest-value transactions.
- Calculate total sales by product category.
- Analyze sales by payment method.
- Determine the highest-performing sales category.
- Identify top sales days.
- Calculate the average transaction amount.
- Count transactions by payment method.
